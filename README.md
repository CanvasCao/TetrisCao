# TetrisCao
plug-in based on canvas

    <script src='tetrisCao.js'></script>
    <script src='underscore.js'></script>
        <script>
            onload=function(){
                var tc=new TetrisCao('box');
            }
        </script>
    
    <body>
    <div id="box"></div>
    </body>

此插件需要导入underscore.js
new TetrisCao()中传入id值就可以新建一个俄罗斯方块游戏 惊了！
